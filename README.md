# What's included?

- Webpack
- TypeScript
- Angular2
- Jade-Templating
- SASS-Styling
- Bootstrap 4 (alpha)

# Setup

- npm install -g webpack webpack-dev-server
- npm install
- npm start
- Browse to localhost:8080
